# Projeto Bibliotech - Monitoramento de Sala com ESP32 e RFID

Sistema IoT de controlo de acesso e lotação de sala com Dashboard Web em tempo real.

## Descrição
O Bibliotech utiliza um ESP32 como servidor web e cérebro lógico. O sistema conta pessoas através de sensores ultrassónicos HC-SR04 e valida a identidade de entrada através de um leitor RFID MFRC522. Os dados são enviados em tempo real para um Dashboard gráfico.
